Подключение тегов <script> и <link> может выполняться как в заголовке так и в теле документа. Выполнение скриптов <script> - определяется атрибутом o5depends независимо от async (а вот defer лучше не использовать). Для библиотечных скриптов и линков рекомендуется указывать адрес относительно предопределённой ссылки '_url_olga5'. При этом вместо src и href предлагается использовать соответственно data-src и data-href

Скрипт o5com!.js должет быть последним, а его адрес д.б. либо абсоютным, либо относительно тега <base>.

Информация о новом скрипте должна добавляться в библиоту в виде труктуры: 

const W = {
    modul: olga5_modul,   // имя класса модуля для регистрации W
    Init: WndInit,        // вызывается при инициализации
    
    
}

По завершению функции Init подключаемого скрипта ею должно быть вызвано событие: 

window.dispatchEvent(new CustomEvent('olga5_sinit', { detail: { modul : W.modul }}))

Все внутренние скриптовые css-описания рекомендуется встраивать непосредственно в js-скрипты.
