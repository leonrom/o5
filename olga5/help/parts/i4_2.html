Подключение зависания к объекту

В атрибуте class тега необходимо указать класс olga5_shp с возможными квалификаторами, разделяемыми символом ':'. Также можно задать дополнительные атрибуты olga5_frames, olga5_owners и olga5_repeat. Например: 

    <div id='div1' class="olga5_shp:Up" olga5_frames="id:shp-demo,C:container:2">

Здесь Up - один из возможных квалификаторов:

  • Top      — блок зависает вверху контейнера;
  • Bottom   — -"- внизу контейнера;
  • Up       — зависание происходит по достижении верха контейнера;
  • Down    — -"- низа контейнера;
  • Shift  — сдвигает предыдущий зависший блок;
  • Push   — сталкивает -"- ;
  • Over   — наезжает на -"- ;
  • i         — целое число,- уровень блока;
  • Alive  — автоматически восстанавливает зависание блока;
  • Kill   — отключает зависание у находящихся выше блоков;
  • None   — отключение событий тега,- обработка не выполняется;

Дополнительные атрибуты olga5_frames и olga5_owners содержат один или несколько наборов полей: код, значение и счётчиков. Обязательное поле код определяет:

  • Id       — произвольный (блочный, не обязательно контейнер) тег с id равным значению;
  • Class — ближайший контейнер, содержащий класс заданный значением;
  • Node  — аналогично контейнер с nodeName равным значению;
  • Scroll — контейнером является счётчик-ный по выложенности скроллируемый контейнер;
  • Bord — блочный контейнер с видимой границей.

Если olga5_owners не указан (или не определён код), то принимается по-умолчанию значение 'B'. Для olga5_frames аналогично принимается значение 'S'.
