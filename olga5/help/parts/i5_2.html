Простейший способ подключения всплытия окна по событию click - добавление тегу атрибута o5popup, значением которого должна быть одна или две подстроки, разделённые символом ';':

<span o5popup='./add/popup.html ; width: 211, height: 211'> показать </span>,

Список параметров в правой подстроке может содержать стандартные параметры окна, а также иметь дополнительные:

  • text - содержит 'текст тега' при открывшемся окне;
  • head - альтернативный заголовок всплывающего окна, при отсутствии значения - мигания не будет;
  • defid - id тега с умалчиваемыми значениями.

При отсутствии у тега атрибута id всплывшее окно исчезает после клика по основному окну.

Для возможности всплытия окна по произвольному событию тега предусмотрена функция PopUp, например

<span onmouseenter="window.olga5.PopUp(this, './add/popup.html', 'width: 211, height: 211')> показать </span>,

Более общий способ всплытия окон - использование функции PopUp(), например:

<span onClick="window.olga5.PopUp(this, './add/popup.html', 'width: 211, height: 211')> показать </span>,

<span onmouseenter="window.olga5.PopUp(this, './add/popup.html', 'width: 211, height: 211')> показать </span>,

Для визуализации всплывающих окон могут автоматически добавляться классы: olga5_popup, olga5_popup_isTag и olga5_popup_isAct.
