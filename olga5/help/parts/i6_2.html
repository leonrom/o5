Для вставки (добавления) содержимого, например тега с id='i3' из файла 'inc1.htm' в документ внутрь <div> с id='d1' последний должен содержать атрибут o5include:

<div id='d1' o5include='./inc1.html?#i3' ></div>

Значение o5include содержит путь к файлу и необязательный селектор тега, разделённые символом '?'. Здесь символ '#' - префикс селектора по атрибуту id. Также префикс м.б. '.' для атрибута class либо отсутствовать,- для селектора по наименованию тега.
