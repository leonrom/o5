Задача: вставить рисунок (размером: ширина=320px и высота=183px):

в блок меньшего размера с заданными граничными значениями высоты и ширины без изменения пропорций рисунка с автоматическим выставлением его размеров.

Результат вставки (числа в желтых полях - можно редактировать) для

<div class="moediv" style= "max-width: px; max-height: px; ">

Здесь стили определены как: 

.moediv {
    width: fit-content;   
    height: fit-content;
    width: -moz-fit-content;
    height: -moz-fit-content;
}
    .moediv img {    
    object-fit: scale-down;
    max-height: inherit;
    max-width: inherit;    
} 
    .moecode {
    background-color: #eeeeee;
    font-family: monospace;
    font-size: x-small;
    padding: 0.8em;
    width:inherit;
    word-wrap: break-word;
    max-width: 90%;
    overflow-x: auto;
}
    .moediv {
    min-width: 33px;
    min-height: 22px;
    background-color: #eeeeee;
    padding: 0;
    margin: 0;
    overflow: auto;
    border: 2px solid blue;
    display: flex;
}
    img {    
    padding: 0 !important;
    margin: 0 !important;
}

где существенными для вписывания изображения являются стили '.moediv' (первый) и '.moediv img'.

Пояснения к полям ввода граничных ширины и высоты:
  • увеличение размеров сверх размера рисунка тут не имеет эффекта;
  • после изменения каждого из значений следует нажать Enter;
  • при вводе недопустимого значения поля окрашиваются красным цветом, а допустимыми значениям являются лишь числа меньше 1000;
  • "Note that when reloading the page, Gecko and IE will ignore the value specified in the HTML source, if the value was changed before the reload"
    (по-простому: после перезагрузки в FireFox'е и IE надо игнорировать числа в желтых полях, а сразу вбивать их по-новому).

Если требуется включение рисунка как ссылки на его оригинальный прототип (например, как рисунки вставляются Blogger'ом), то тег img следует заключить в тег a (а не в div). Например вот так: 

<a class="moediv" 
style="max-width:222px; max-height:111px; padding:0; text-align:left;"
href ="https://1.bp.blogspot.com/-lQitJn5pexE/WstcYx5OIOI/AAAAAAAABlU/GYmUunWQjZwcZFt9eISSz-zUYOpkpxZIACPcBGAYYCw/s428/o1.png"
>
    <img src="https://1.bp.blogspot.com/-lQitJn5pexE/WstcYx5OIOI/AAAAAAAABlU/GYmUunWQjZwcZFt9eISSz-zUYOpkpxZIACPcBGAYYCw/s320/o1.png" 
    />
</a>
