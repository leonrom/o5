Примеры пересчета параметров по ссылочному аргументу.
умолчание: 

	text: '', group: '', head: '', 
	width: 588, height: 345, top: 11, left: -22, 
	alwaysRaised: 1, alwaysOnTop: 1, menubar: 0, toolbar: 0, status: 0, resizable: 1, scrollbars: 0,innerwidth: '', innerheight: '', screenx: '', screeny: ''
Результат показывается в поле 'Рез:' после клика по примеру

  1. ...PopUp('', './popup.html', '')
    - параметры по умолчанию

  1. ...PopUp('', './popup.html', 't =1, left: 199; width=222, h:111; group=1')
    p1 - базовый для остальных (его недостающее - из умалчиваемых значений)

  2. ...PopUp('', './popup.html', 'width=77, left=33, p1, p111, g:2')
    p2 недостающее - из p1, ссылка на p111 - ошибочна (см. протокол);

  3. ...PopUp('', './popup.html', 'width=77, p1, p2')
    p3 недостающее - из p1 и потом p2;

  4. ...PopUp('', './popup.html', 'width=88, p2, p1')
    p4 недостающее - из p2 и потом p1,- от предыдущего отличается left и group;

Результирующие значения top могут отличаться от заданных на десяток пикселов, из-за учета высоты заголовка окна
