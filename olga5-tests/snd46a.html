Примеры подключения <img> с вызовом всех модулей ядра библиотеки.
Для звучания - кликнуть по иконке (или навести на неё - для 4, 5 и 6<a>)

проверка с тегом <A>

<a class='olga5_snd:o:href' target="olga5-addPage" href="sounds+/Ceza1-25.mp3" > Ceza1-25.mp3 </a>
<a class='olga5_snd:o:a:href' target="olga5-addPage" href="sounds+/Ceza1-25.mp3" > Ceza1-25.mp3 </a>
<a class='olga5_snd:href' target="olga5-addPage" href="sounds+/Ceza1-25.mp3" > Ceza1-25.mp3 </a>
<a class='olga5_snd:a:href' target="olga5-addPage" href="sounds+/Ceza1-25.mp3" > Ceza1-25.mp3 </a>

проверка с вариантами ссылок href - загрузка аудио

<a class='olga5_snd:o:_href' target="olga5-addPage" _href="sounds+/Ceza1-25.mp3" > Ceza1-25.mp3 </a>
<a class='olga5_snd:o:href' target="olga5-addPage" _href="sounds+/Ceza1-25.mp3" > Ceza1-25.mp3 </a>
<a class='olga5_snd:o:data-href' target="olga5-addPage" data-href="sounds+/Ceza1-25.mp3" > Ceza1-25.mp3 </a>
<a class='olga5_snd:o:href' target="olga5-addPage" data-href="sounds+/Ceza1-25.mp3" > Ceza1-25.mp3 </a>

проверка с вариантами ссылок src - загрузка image_play

<img class="olga5_snd:o:Ceza2-25" src="images+/play.png" image_play=src/>
<span class="olga5_snd:o:src" src="+Ceza2-25"/>...<span>
