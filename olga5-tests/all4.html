Примеры выборочной обработки тегов через QuerySelectorAll.

Показаны 4 пары иконок. В зависимости от атрибута class='olga5_Start' иконки могут быть последовательно (согласно очерёдности в <head>) обработаны скриптами

  • o5snd - теги с 'class='olga5_snd'' и даёт 'красную' картинку
  • o5ref - теги с атрибутом '_src' даёт 'чёрную ' картинку

Первая в паре иконка имеет class='olga5_snd' и может обрабатываться скриптом o5snd, а вторая - может измениться последующим скриптом o5ref.


1. задано class='olga5_Start' - обрабатывается o5snd и (потом) o5ref
красная — ; чёрная — ;

2. задано class='olga5_Start:o5ref:o5snd' - то же самое, что и 1
красная — ; чёрная — ;

3. задано class='olga5_Start:o5snd' - обрабатывается только o5snd
красная — ; нету — ;

4. задано class='olga5_Start:o5ref' - обрабатывается только o5ref
нету — ; чёрная — ;

5. НЕ задано class='olga5_Start' - НЕ обрабатывается ничем
нету — ; нету — ;
