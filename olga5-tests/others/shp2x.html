секция <head>
<head>
<meta http-equiv="content-type" content="text/html; charset=UTF-8">
<title>olga5 поотдельности</title>
<link rel="shortcut icon" href="./favicon.svg" type="image/x-icon">

<link href='+o5ini.css' media='screen' rel='stylesheet'/>

<script data-src='+o5ref.js'
o5tag_attrs='img= src, _src' 
o5urlrfs="
	музоны=_url_olga5 + ../blog/media; tt = музоны + /tt;
	btnSound = '_url_olga5 + ../blog/media/image/play.png'
"
></script>
<script _src='+o5shp.js'></script>
<script data-src='+o5snd.js'
кнопка_stop = '_url_olga5 + ../blog/media/image/myMusikIT.png'
></script>
<script src='../olga5/o5com!.js' o5debug=0></script>
</head>

Посмотреть протокол работы примера можно перейдя по ссылке и, в открывшемся окне, вызвав отладчик браузера. Аналогичные примеры с различными вариантами подключения скомпилированных скриптов можно посмотреть тут: all2_, all2a, all2c, all2cc (краткие пояснения приведены на страницах примеров).
