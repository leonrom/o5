Примеры ссылок на атрибуты. Для звучания - навести (после активации страницы)
курсор на текст 'курсор сюда' в тегах <a>, <span> или на иконку

проверки с тегом <A> - переход по ссылке на href после клика на паузе

<a class='olga5_snd:href' target="add" href="sounds+/Ceza1-25.mp3" > кликнуть тут </a>
<a class='olga5_snd:a:href' target="add" href="sounds+/Ceza1-25.mp3" > кликнуть тут </a>
<a class='olga5_snd:o:href' target="add" href="sounds+/Ceza1-25.mp3" > курсор сюда </a>
<a class='olga5_snd:o:a:href' target="add" href="sounds+/Ceza1-25.mp3" > курсор сюда </a>

проверки с вариантами ссылок _href и data-href

<a class='olga5_snd:o:href' target="add" _href="sounds+/Ceza1-25.mp3" > курсор сюда </a>
<a class='olga5_snd:o:_href' target="add" _href="sounds+/Ceza1-25.mp3" > курсор сюда </a>
<a class='olga5_snd:o:href' target="add" data-href="sounds+/Ceza1-25.mp3" > курсор сюда </a>
<a class='olga5_snd:o:data-href' target="add" data-href="sounds+/Ceza1-25.mp3" > курсор сюда </a>

проверки с вариантами ссылок src в тегах span и img

<span class="olga5_snd:o:src" src=+Ceza2-25/>курсор сюда<span>
<img class="olga5_snd:o:Ceza2-25" src="images+/play.png" image_play=src/>
