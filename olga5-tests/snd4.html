Примеры подключения <img> с вызовом всех модулей ядра библиотеки.
Для звучания - кликнуть по иконке (или навести на неё - для 4, 5 и 6<a>)

Ссылки на аудиофайл содержащие пробелы д.б. записана в кавычках, как в 2, 3 и 5 (с вольностями в задании квалификаторов у класса olga5_snd). В примере 6<a> показано использование атрибута href для ссылки на аудиофайл. При клике в состоянии 'pause' выполнится переход по ссылке
Иконка меняется (при проигрывании) только для 3 и 4, у которых (по-разному) заданы image_play. Проигрывание иконок с квалификатором Over выполняется лишь после активации страницы, Чтобы отсановить звучание иконок с квалификатором Alive следует повторно кликнуть для паузы и потом увести курсор.
В примерах 1, 2, 3, 4 показаны различные вариантты подключения аудио-файла Ceza2-25.mp3

1 <img class="olga5_snd:+Ceza1-25" src= "+myTunes-icon"/>
    простейший пример для <img>,- звучание по клику, прекращение по уходу курсора.
2 <img class="olga5_snd:alive:'  +   Ceza1-25  '" />
    не прерывается по уходу курсора; src не указан,- берутся btn-stop и btn-play
3 <img class="olga5_snd:L:A:Ceza1-25" src= "+myTunes-icon" image_play=".../Spotify-icon.png"/>
    зацикленное звучание по клику до следующего клика ('L'= 'loop', 'A'= 'alive')
4 <img class="olga5_snd:over:sounds+Ceza2-25.mp3" src="images+ /solfedjio.png" image_play=btn_play/>
    однократное звучание (с изменением иконки) по наведению курсора.
5 <img class='dlassaaa:A olga5_snd:over : aЯё: loop  :   "   sounds + Ceza1-25.mp3" ' src=solfedjio/>
    зацикленное звучание по наведению курсора, не прерывается с его уходом.
6 <a>

<b> 		<a class='olga5_snd:href' target="olga5-addPage" href="sounds+/Ceza1-25.mp3" > кликнуть тут </a>

Использовались следующие 'кнопки' (эти ниже - некликабельные):
— скриптовый параметр btn-stop (icons8-youtube-music-48.png)
— скриптовый параметр btn-play (icons8-youtube-music-50.png)
— теговый атрибут image_play=imageplay (play.png)
— файл solfedjio.png

Дополнительно показаны варианты подключения стандартных плейеров:
здесь: class='olga5_snd:H' - игнорируется в o5snd
<audio _src="sounds+/Ceza1-25.mp3">
атрибут _src будет обработан модулем o5ref
и 'сразу' заменит src. 		здесь: class - не задан,
<audio data-src=  "sounds+ Ceza1-25.mp3">.

Здесь подключение o5snd.js имеет вид: 

<script async src='+o5snd.js'
    o5urlrfs="
        Ceza1-25=sounds+/Ceza1-25.mp3;
        Ceza2-25=sounds+/Ceza2-25.mp3;
        imageplay=images + /play.png;
        solfedjio=images+ /solfedjio.png;
    "
    btn_stop='images + /icons8-youtube-music-48.png'
    btn_play='images + /icons8-youtube-music-50.png'
/></script/>

Тут отсутствует иконка — myTunes-icon  ,- она задана (для тестирования) в скриптовом атрибуте o5urlrfs подключения o5ref.src

Примечание: сообщение об ошибке "o5ref: неопределённые адреса" в console.log выдаетя потому, что те адреса определены локально для модуля o5snd и недоступны модулю o5ref.
