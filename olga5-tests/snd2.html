Примеры задания атрибутов тега <img> :

1 Заданы: src для без звука - 'imagestop', а для звучание - image_play

<img id="1" src=imagestop image_play=imageplay class="olga5_snd:аудио_файл" />

2 без звука как в 1, но при звучании иконка не меняется (т.к. не задан image_play)

<img id="2" src=imagestop class="olga5_snd:аудио_файл" />

3 То же, что 2, при этом image_play задан равным _src

<img id="3" _src=imagestop image_play=_src class="olga5_snd:аудио_файл" />

4 Не задан src, поэтому без звука берётся из btn-stop (а звучание - image_play)

<img id="4" image_play=imageplay class="olga5_snd:аудио_файл" />

5 Не заданы ни без звука, ни звучание,- берутся, соответственно, из btn_stop и btn_play.

<img id="5" class="olga5_snd:аудио_файл" />


Дополнительно: ссылка на аудио_файл в качестве квалификатора audio_play будет взят href

<a href=аудио_файл class="olga5_snd:over"/>аудио_файл</a/>


Использовались следующие 'кнопки' (эти ниже - некликабельные):
— теговый атрибут src=imagestop;
— теговый атрибут image_play=imageplay;
— скриптовый параметр btn-stop;
— скриптовый параметр btn-play.


Здесь подключение o5snd.js имеет вид: 

<script async src='../olga5/o5snd.js'
    btn_stop='images + /icons8-youtube-music-48.png';
    btn_play='images + /icons8-youtube-music-50.png';
    imagestop='_url_olga5 + ../blog/media/image/myMusikIT.png';
    imageplay='images + /play.png';
    аудио_файл=sounds + /Ceza1-25.mp3;
/></script/>
<script async src='../olga5/o5com!.js' o5debug=0
    images="_url_olga5 + ../blog/media/image"
    sounds=https://archive.org/download/clouds_20200310;
/></script/>
