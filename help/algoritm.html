Состояния объектов

Подвисабельные объекты могут иметь следующие признаки состояния (задаются в aO5.act):

  • uScroll — тег будет обрабатываться при скроллинге,-. т.е. он может наезжать на зафксированные (если у него самого неопределён xFixed). Изменяется IntersectionObserver'ом в модуле DoInit для нефиксированных объектов.
  • canFix — тег может фиксироваться на границе контейнера только если перед этим он был полностью видан на экране.
  • xFixed — фиксированность объекта на контейнере, принимающая значенияё:
    • null - не фиксирован
    • true - фиксирован при 'наезде' на другой зафиксированный объект.
    • tag - фиксирован при 'вылете' за границы контейнера <tag>.
    Пересчитывается при каждом сколлинге (если включен контроль скроллирования) для всех объектов с установленным uScroll.
  • oldIR — предыдущая видимость тега (т.е. значение intersectionRatio)

Модуль DoInit

Модуль контролирует поведеник подвисабельных тегов на эране:

  • создаёт обозреватель obsrv0 с обработчиком Obsrv0();
  • подключает к обозревателю obsrv0 все теги, содержащие класс 'olga5_shp' при отсутствии (у этого класса) квалификатора ':N' (т.е. 'none')

Обработчик DoInit.Obsrv0() выполняет:

    если увеличилась видимость тега (т.е. intersectionRatio > oldIR)
    • Если aO5shp не определён (т.е. это первое увеличение с нулевого значения)
      • инициирует тег: создаёт в нём объект aO5shp;
      • добавляет aO5shp в массив wshp.aO5s.
    • устанавливается для тега (точнее, для его aO5shp.act) признак uScroll=true.
    • если тег стал полностью видимым (т.е. intersectionRatio==1), то устанавливает для него признак canFix=true.
    если уменшилась видимость тега (т.е. intersectionRatio < oldIR
    • если canFix и уменшение видимости произошло 'в направлении' cls.dirV то включается контроль скроллинга
      • иначе
    • если тег стал полностью НЕвидимым (т.е. intersectionRatio==0)
      • задаются uScroll=false (тег исключается из обработки при скроллинге) и canFix=false;
      • ?? если среди всех (в wshp.aO5s) тегов отсутствуют с canFix, то отключается контроль скроллинга.
     запоминает oldIR =intersectionRatio.
